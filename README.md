# gulp_webpack
# 1.安装nodejs,详细请到官网：https://nodejs.org/
# 2.安装gulp, 具体步骤如下：
  npm install gulp -g
# 3.安装webpack，具体步骤如下：
  npm install webpack -g
# 4.初始化依赖组件：
  npm install

# 5.使用gulp构建css的合并和压缩，使用webpack构建js
# 6.webpack能更好的做模块化和模块依赖分析
# 7.一个命令行完成发布相关事宜，压缩，合并，按需加载模块依赖管理，版本号管理
# 8.gulp build 将会构建线上代码，而gulp build --debug将构建调试代码
